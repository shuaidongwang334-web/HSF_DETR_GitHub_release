# RTTS split files

These files were generated from the uploaded RTTS label folders.

- `RTTS_train.txt`, `RTTS_val.txt`, `RTTS_test.txt`: image paths using the common `.jpg` extension assumption.
- `RTTS_*_image_stems.txt`: image stems without extension.
- `RTTS_*_label_files.txt`: label file paths.

RTTS was not split using a random seed. The exact split files are provided for reproducibility.
